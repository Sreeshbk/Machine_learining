# Machine_learning

## Clone Workspace

```
git clone https://github.com/Sreeshbk/Machine_learning.git
```

## Running pyspark

```
docker run -p 8888:8888 --add-host=host.docker.internal:host-gateway jupyter/pyspark-notebook
```

## Install Packages

```
pip install hyperopt mlflow
```
